# Emoji Calculator

A fun, interactive web application built with Flask that translates mathematical expressions into moods. 

Instead of typing numbers, users click on various emojis that represent digits and operators. The application evaluates the hidden mathematical expression, calculates a mood score based on the result, and generates a short creative story explaining the "mood" of the numbers.

## Requirements
- Python 3
- Flask
- emoji

## How to Run
1. Install dependencies: `pip install flask emoji`
2. Run the server: `python3 hello.py`
3. Access the application in your browser at `http://127.0.0.1:5001`
